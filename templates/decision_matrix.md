# Matrice de Décision : Cloud vs On-Premise vs Hybride

| Critère | Cloud Public | On-Premise | Hybride |
| :--- | :--- | :--- | :--- |
| **Coût (FinOps)** | OPEX, Variable | CAPEX, Fixe | Mixte |
| **Souveraineté** | Risque Cloud Act  | Total contrôle  | Partiel |
| **Green IT** | Efficacité mutualisée | Contrôle direct | Optimisé par charge |
| **Scalabilité** | Immédiate [cite: 16] | Limitée | Flexible  |
